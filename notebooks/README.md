# Notebooks

This directory contains Jupyter notebooks for data exploration, model prototyping, and visualization.

## Planned Notebooks

| Notebook | Description |
|----------|-------------|
| `01_dem_exploration.ipynb` | Explore DEM data, visualize elevation and slope |
| `02_flood_simulation.ipynb` | Prototype flood models, tune parameters |
| `03_population_analysis.ipynb` | Analyze building footprints and population clustering |
| `04_risk_scoring.ipynb` | Multi-criteria risk scoring with sensitivity analysis |
| `05_pathfinding_demo.ipynb` | A* pathfinding visualization with spline smoothing |
| `06_resource_optimization.ipynb` | Resource allocation scenarios and coverage analysis |

## Setup

```bash
pip install jupyter matplotlib folium geopandas
jupyter notebook
```
